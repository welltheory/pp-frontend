### Development setup
1. Make sure you have NodeJS installed on your machine. If not, you can download it [here](https://nodejs.org/en/download/) or install it using [Homebrew](https://brew.sh/).
2. Clone this repository to your local machine.
3. `cd` into `./client` and run `npm install` to install the dependencies.
4. Run `npm start` to start the development server.
